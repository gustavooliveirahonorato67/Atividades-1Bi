O que é Refatoração Contínua?
A refatoração contínua é o processo de melhorar a estrutura interna de um código sem alterar o comportamento externo. Em vez de esperar o projeto ficar "pesado" para fazer uma grande limpeza, o desenvolvedor melhora o código aos poucos. O foco é transformar nomes genéricos em nomes claros e organizar a lógica para que qualquer pessoa do time entenda o que o código faz rapidamente.

Código Sem Refatoração
fun main() {
    val l = listOf("Kotlin", "Java", "Swift")
    for (i in l) {
        println("Eu gosto de programar em $i")
    }
}

Código Refatorado
fun main() {
    val linguagens = listOf("Kotlin", "Java", "Swift")
    exibirPreferencias(linguagens)
}
fun exibirPreferencias(lista: List<String>) {
    for (linguagem in lista) {
        // Aqui o código imprime cada uma: Kotlin, depois Java, depois Swift
        println("Eu gosto de programar em $linguagem")
    }
}
