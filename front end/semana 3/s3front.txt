1. display: grid
Essa propriedade transforma o elemento pai em uma grade. Ela permite que de pra dividir o layout em linhas e colunas, funcionando como uma tabela invisível que organiza onde cada bloco do site deve ficar.


2. grid-template-areas
Ela permite que nomeie as áreas do layout (como menu e cabecalho). É como se criasse um mapa visual com as palavras, facilitando muito o entendimento de onde cada parte do site tá posicionada.

3. align-items: center
Usada dentro de um display: flex, essa propriedade alinha os elementos verticalmente ao centro. No cabeçalho, ela garante que o título e o botão fiquem perfeitamente alinhados na mesma linha, sem um ficar mais alto que o outro.


4. transition: background-color 0.3s
Ela define o tempo que uma mudança leva para acontecer. Quando você passa o mouse no botão, em vez de a cor mudar instantaneamente, ela leva 0.3 segundos para se transformar, criando um efeito visual menos seco e mais suave.

5. @media (max-width: 768px)
Essa é a base do design responsivo. Ela cria uma regra que só é ativada quando a tela é menor que 768 pixels (como em celulares). Nesse caso ai do exercício, ela foi usada pra esconder o menu lateral e empilhar o conteúdo para caber em telas mais pequenas.
